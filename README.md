# AI Novel Assistant

AI驱动的小说创作协作工具

## 功能特性

- 🤖 AI协作创作：智能完善角色、设定、章节内容
- 📚 项目管理：完整的小说项目组织和管理
- 🔍 一致性检查：实时检测角色、设定、情节一致性
- 📖 章节编辑：专业的编辑器体验
- 📤 多格式导出：支持EPUB、TXT等格式

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 数据库设置
```bash
npm run db:push
```

## 技术栈

- **后端**: Node.js + Express + Prisma + SQLite
- **前端**: Vue 3 + Quasar + Monaco Editor
- **AI服务**: OpenAI API
- **桌面**: Electron (计划中)
